eCommerce Single Page Application

Project Description

A single page application prototype of an eCommerse site. Demo

Project Details

  • Prototype includes essential features of an eCommerce site such as product list browsing, guest cart, and checkout.
  • Front End is built with React, React-Router, Redux state management, and a Bootstrap 3 HTML template from Themeforest.
  • RESTful API is built with Node/Express and MongoDB/Mongoose.
  • Axios library is used to help with fetching resources and Web LocalStorage API is used to build the guest car functionality.
  • Learned at lot about the react ecosystem and how web technologies are used together for fetching resources, client and server routing, and client state management.
